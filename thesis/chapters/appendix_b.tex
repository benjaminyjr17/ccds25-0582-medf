\chapter{Framework YAML Definitions}
\label{app:yaml}

This appendix reproduces excerpts from the YAML configuration files that define the three governance frameworks integrated into MEDF. These files are located in the \texttt{app/frameworks/} directory of the repository. The excerpts below are verified against the actual source files as of the current repository version.

\section{EU ALTAI (\texttt{eu\_altai.yaml})}

The EU ALTAI framework defines six criteria mapped to the Unified Dimensions, with a total of 22 requirements and assessment questions. The dimension weights reflect the structural emphasis of the ALTAI guidance, with Safety and Robustness receiving the highest weight (0.22) due to its five requirements.

\begin{lstlisting}[language={}, caption={EU ALTAI framework definition (excerpt, verified against source).}]
id: eu_altai
name: EU Assessment List for Trustworthy AI (ALTAI)
version: "2020"
description: Baseline criteria distilled from EU ALTAI guidance.
criteria:
  - id: altai_transparency
    dimension: transparency_explainability
    requirements: [ALTAI-TR-1, ALTAI-TR-2]
    weight: 0.12
  - id: altai_fairness
    dimension: fairness_nondiscrimination
    requirements: [ALTAI-FR-1, ALTAI-FR-2, ALTAI-FR-3]
    weight: 0.14
  - id: altai_safety
    dimension: safety_robustness
    requirements: [ALTAI-SF-1, ..., ALTAI-SF-5]
    weight: 0.22
  - id: altai_privacy
    dimension: privacy_data_governance
    requirements: [ALTAI-PR-1, ..., ALTAI-PR-5]
    weight: 0.20
  - id: altai_human_oversight
    dimension: human_agency_oversight
    requirements: [ALTAI-HO-1, ..., ALTAI-HO-4]
    weight: 0.18
  - id: altai_accountability
    dimension: accountability
    requirements: [ALTAI-AC-1, ALTAI-AC-2, ALTAI-AC-3]
    weight: 0.14
\end{lstlisting}

\noindent The weights sum to 1.00: $0.12 + 0.14 + 0.22 + 0.20 + 0.18 + 0.14 = 1.00$.

\section{NIST AI RMF (\texttt{nist\_ai\_rmf.yaml})}

The NIST AI Risk Management Framework defines 18 subcategories mapped to the six Unified Dimensions. The weights reflect the emphasis of the NIST RMF's four core functions (Govern, Map, Measure, Manage) on each dimension.
